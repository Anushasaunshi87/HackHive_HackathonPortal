# 🚀 Hackathon Discovery Platform  

## 📌 Problem Statement  
Students struggle to find relevant hackathons because information is scattered across multiple platforms. There is no centralized system that allows easy discovery and filtering based on domains or interests. This makes it difficult for students to identify and participate in suitable hackathons.

## 🎯 Objective  
To build a web platform that aggregates hackathon information and allows students to filter events based on their domain, skills, and interests.

## ❗ Challenges  
- Hackathon details are spread across different websites and communities  
- Lack of domain-specific filtering (e.g., AI, Web Dev, Cybersecurity)  
- Time-consuming process to search and compare events  

## 💡 Proposed Solution  
Develop a user-friendly web application where students can:  
- Browse upcoming hackathons in one place  
- Filter hackathons by domain, date, and skill level  
- Access direct registration links  

## 🛠️ Tech Stack (Example)  
- Frontend: HTML, CSS, JavaScript / React  
- Backend: Node.js / Django  
- Database: MongoDB / MySQL
